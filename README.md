# Project-Spoken # SPOKEN
Chat log organizer that converts messy conversations into clean speech bubbles.
SHOKOLABO의 첫번째 프로젝트입니다. 

복잡한 채팅 로그를 읽기 좋은 대화 형태로 정리해주는 웹 도구입니다.
불필요한 요소를 제거하고,
말풍선 형식으로 보기 좋게 정돈해줍니다. <br/>
협업 및 에러 제보 기타 문의는 sususmumichinori@gmail.com 로 연락주세요

## ✨ 주요 기능

- 같은 사람 대사 자동 합치기
- 시간 표시 제거
- ㅋㅋ / ㅎㅎ 정리
- 말풍선 형태로 변환
- 브라우저 내에서만 동작 (데이터 저장 없음)

## 🔒 개인정보 안내

입력한 대화 내용은 서버에 저장되지 않습니다.  
모든 처리는 사용자의 브라우저에서만 이루어집니다.
데이터는 관리자도 알 수 없습니다. 필요시 반드시 저장해주세요! 

## 🚀 사용 방법

1. 대화를 복사해서 입력창에 붙여넣기
2. 원하는 옵션 선택
3. "정리하기" 버튼 클릭
